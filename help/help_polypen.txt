Polypen Help (click to hide help):

DRAWING:
- CTRL+LMB: insert geometry
- SELECTMOUSE: select vertex, edge, or face
- A: deselect all

MERGE / DELETE / DISSOLVE:
- CTRL+D: dissolve selected edge
- X: delete selection

CONFIRM / CANCEL:
- ENTER / RETURN: confirm changes, return to Edit Mode
- ESC: cancel operation

TIPs:
- Creating vertices/edges/faces is dependent on your selection:
-- select an edge then CTRL+LMB to create a triangle from that edge, LMB again to turn it into a quad
