# RetopoFlow

Retopoflow is a suite of fun, sketch-based [retopology](https://en.wikipedia.org/wiki/Retopology) tools for [Blender](https://www.blender.org/) from [Orange Turbine](https://orangeturbine.com) that generate geometry which snaps to your high poly objects as you draw on the surface.

![Blender Market](help/v4/images/bears.jpg)

You can read about the features and purchase a copy on [Superhive](https://superhivemarket.com/products/retopoflow?ref=806) (formerly Blender Market).

[The documentation](https://docs.retopoflow.com/) covers the installation and usage of all tools included in the add-on.


## Support

If you find an issue with Retopoflow, please send us a message via the Superhive creator support inbox. You can also open an issue here on GitHub.

Please provide as much detail as possible, including screenshots, a copy/paste of any error message, and an example file.


## Contributing

Pull requests are welcome!
If you'd like to contribute to the project then simply fork the repo, work on your changes, and then submit a pull request.
We are quite strict on what we allow in, but all suggestions are welcome.
If you're unsure what to contribute, then look at the [open issues](https://github.com/CGCookie/retopoflow/issues) for the current to-dos.
